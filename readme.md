## Industry Tool to Previous CourseWork Mapping

| Tool | Coursework | Purpose/Benefits |
|------|------------|-----------------|
| FastAPI | Flask | Faster, API documentation built-in |
| Docker | Virtual Machines | Faster, portable |
| Git/GitHub | Saving files manually | Documents changes, team collaboration |
| GitHub Actions | Running Scripts Manually | Testing and deployment on every commit |

Academic tools are used for teaching concepts and theory clearly. However, industry tools are more applicable to real-life work fields due to their scalability, efficiency, and ability to work in group environments.
